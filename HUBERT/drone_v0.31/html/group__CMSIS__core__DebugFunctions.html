static

#include <cmsis/core_cm4.h>

ITM Check Character.

This function checks external variable ITM_RxBuffer whether a character is available or not. It returns '1' if a character is available and '0' if no character is available.

Returns
0 No character available
1 Character available
static __INLINE int32_t ITM_ReceiveChar ( void  )
static

#include <cmsis/core_cm4.h>

ITM Receive Character.

This function inputs a character via external variable ITM_RxBuffer. It just returns when no debugger is connected that has booked the output. It is blocking when a debugger is connected, but the previous character send is not transmitted.

Returns
Received character
-1 No character received

ITM Send Character.

This function transmits a character via the ITM channel 0. It just returns when no debugger is connected that has booked the output. It is blocking when a debugger is connected, but the previous character send is not transmitted.

Parameters
